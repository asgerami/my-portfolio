# My Portfolio

A simple portfolio I built with [Astro](https://astro.build) and [TailwindCSS](https://tailwindcss.com), focused on performance and clean architecture.

## Overview

- **Stack**: Astro, TailwindCSS, TypeScript.
- **Performance**: Static site generation for optimal loading speeds and perfect LightHouse scores.
- **Design**: Minimalist, responsive UI with dark mode support.

## Development

### Prerequisites

- Node.js 18+
- npm

### Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The application will be available at `http://localhost:4321`.

## Deployment

Designed for static deployment on platforms like Vercel or Netlify.

```bash
# Build for production
npm run build
```
