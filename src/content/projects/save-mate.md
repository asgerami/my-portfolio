A powerful financial management app designed to help users achieve their financial goals efficiently. Save-Mate provides comprehensive budgeting tools and expense tracking to take control of your finances.

## Key Features

- **Income and Expense Tracking** - Easily log and monitor your financial transactions
- **History Section** - Group transactions by month or year for comprehensive financial overview
- **Animated Tooltips** - Enhanced user experience with engaging and informative guidance
- **Simple Transaction Form** - Quick transaction logging with category and date selection
- **Flexible Category Creation** - Manage categories on dedicated page or create on-the-fly
- **Comprehensive Transaction Table** - View detailed data with filtering by date, category, and type
- **Currency Management** - Switch currencies and manage categories seamlessly

## Technical Highlights

The application is built with modern technologies:

- **Next.js** for server-side rendering and optimal performance
- **TypeScript** for type-safe development throughout the application
- **Prisma ORM** with PostgreSQL for robust data management
- **shadcn UI** for beautiful, accessible component library
- **React Hook Form** with Zod validation for efficient form handling
- **TanStack Query** for powerful data fetching and caching
- **Tailwind CSS** for utility-first styling

## Problem Solved

Save-Mate addresses the common challenges of:

- **User Experience** - Creating an intuitive interface for financial management
